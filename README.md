# FiveM Server Status

Simple free FiveM server status into a Discord channel

## Setup

1. Download the latest release
2. Extract Zip File
3. Setup 'botconfig.js' and 'index.js'

## Running

1. ```npm i```
2. ```npm start``` or ```node index.js```
